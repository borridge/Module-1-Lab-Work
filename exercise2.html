Lab Steps

  1. Open (or create) your Module1 folder in VS Code.
  2. Open the Terminal and create a folder called LabExercises:
    mkdir LabExercises
  3. Change directory into LabExercises:
    cd LabExercises
  4. Create a new file called exercise2.html:
    New-Item exercise2.html
  5. In exercise2.html, type an exclamation mark and press Tab to create an HTML skeleton:
    ! + press Tab
  6. Inside the <body> section, add the steps of this lab as HTML content.
  7. Go Live to view and test the HTML:
    (Right-click on the HTML file and select Open with Live Server)
  8. Install Prettier extension and format your code:
    Ctrl + K, Ctrl + F
  9. Read the MDN article for tips:
    Dealing with files (MDN)
